# 安全生产管理平台 Web


## 项目简介

安全生产双重预防管理系统

平台围绕安全生产日常管理场景进行页面设计与模块组织，覆盖隐患治理、隐患上报、日常巡检、教育培训、考核管理、台账查询、统计分析、数据大屏展示等典型业务页面。
符合应急管理部下发的各类关于安全生产方向的政策文件

请注意：

- 本仓库为**展示版仓库**
- 仅用于**产品宣传、界面演示、技术展示**
- **不包含**完整后端服务、数据库脚本、生产部署方案和完整业务闭环

## 主要展示模块

### 1. 平台首页 / 大屏数据看板
展示平台概览、指标汇总、趋势分析和风险分布等可视化内容，用于体现系统首页的信息组织能力与大屏风格设计能力。

### 2. 隐患治理类页面
展示列表查询、条件筛选、详情查看、状态展示、流程跟踪等典型业务交互，体现中后台复杂信息页面的组织能力。

### 3. 教育培训类页面
展示培训计划、培训记录、培训统计等页面结构，体现业务模块化设计和统一视觉语言。

### 4. 考核管理类页面
展示考核台账、统计分析、状态流转、结果查看等页面元素，适合体现多字段表格与复杂表单的交互组织能力。

### 5. 隐患上报页面
通过手机移动端上报隐患进行整改验收闭环/多人协同消除企业日常隐患情况。

### 6. 日常巡检页面
后台PC端制定隐患巡检任务及巡检项，实现按照时间周期为节点的自动化巡检任务派发，生成巡检记录进行归档。

### 7. 后续计划
监管系统/八大作业票/人员定位/人员聚集报警/数据对接上报园区/市局/省厅/重大危险源数据接入等符合危险化学品企业的相关内容。
